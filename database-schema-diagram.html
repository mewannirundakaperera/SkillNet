users

Private user data collection

Core Fields

uid String Firebase Auth UID (Document ID)
email String User's email address
displayName String User's display name
role String User role (user/admin)
createdAt Timestamp Account creation date
updatedAt Timestamp Last update date

Relationships

1:1 userProfiles collection
1:Many requests collection (as owner)
1:Many groups collection (as member)

Security Rules

allow read, write: if isAuthenticated() && isOwner(userId)

userProfiles

Public user profile information

Core Fields

uid String Firebase Auth UID (Document ID)
displayName String Public display name
bio String User biography
skills Array List of user skills
interests Array List of user interests
avatar String Profile picture URL
stats Object Rating statistics

Relationships

1:1 users collection

Security Rules

allow read: if isAuthenticated()
allow write: if isAuthenticated() && isOwner(userId)

requests

One-to-one learning requests

Core Fields

userId String Request owner ID
title String Request title
description String Request description
status String Request status
category String Request category
paymentAmount String Payment amount
participants Array List of participants
responses Array List of responses

Status Flow

draft
open
active
completed

Relationships

Many:1 users collection (owner)
1:Many requestResponses collection

Security Rules

allow read: if isAuthenticated()
allow create: if isAuthenticated() && isRequestCreateOwner()
allow update: if isAuthenticated() && (isRequestOwner() || isAdmin())

grouprequests

Group learning requests

Core Fields

userId String Request creator ID
targetGroupId String Target group ID
title String Request title
description String Request description
status String Request status
votes Array List of user votes
participants Array List of participants
teachers Array List of teachers
selectedTeacher String Selected teacher ID
paidParticipants Array List of paid participants

Status Flow

pending
voting_open
accepted
funding
payment_complete
in_progress
completed

Relationships

Many:1 users collection (creator)
Many:1 groups collection (target)
Many:1 users collection (selected teacher)

Security Rules

allow read: if true (public)
allow create: if isAuthenticated() && isRequestCreateOwner()
allow update: if isAuthenticated() && (isRequestOwner() || isAdmin())

groups

Learning groups and communities

Core Fields

name String Group name
description String Group description
category String Group category
createdBy String Group creator ID
members Array List of member IDs
hiddenMembers Array List of hidden member IDs
memberCount Number Total member count
isPublic Boolean Public visibility flag
image String Group image URL

Subcollections

messages/{messageId} Collection Group chat messages
activeUsers/{userId} Collection Currently active users
typingStatus/{userId} Collection User typing indicators

Relationships

Many:1 users collection (creator)
Many:Many users collection (members)
1:Many grouprequests collection

Security Rules

allow read: if isAuthenticated() && (isPublic || isMember || isAdmin)
allow create: if isAuthenticated() && isOwner
allow update: if isAuthenticated() && (isOwner || isAdmin)

chats

Direct messaging system

Core Fields

participants Array List of participant IDs
lastMessage Object Last message details
createdAt Timestamp Chat creation date

Subcollections

messages/{messageId} Collection Chat messages

Relationships

Many:Many users collection (participants)

Security Rules

allow read, write: if isAuthenticated() && isParticipant

meetings

Video meeting management

Core Fields

hostUserId String Meeting host ID
participants Array List of participant IDs
roomId String Jitsi room ID
meetingLink String Meeting URL
status String Meeting status
startTime Timestamp Meeting start time
endTime Timestamp Meeting end time

Relationships

Many:1 users collection (host)
Many:Many users collection (participants)

Security Rules

allow read: if isAuthenticated() && (isHost || isParticipant || isAdmin)
allow create: if isAuthenticated()
allow update: if isAuthenticated() && (isHost || isAdmin)
